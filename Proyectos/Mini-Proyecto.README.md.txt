# 🧑‍🎓 Registro de Alumnos - Proyecto en Java

Este mini proyecto en Java simula un pequeño sistema para registrar y gestionar información de un alumno.  
Forma parte de mi aprendizaje de fundamentos de programación orientado a buscar mi primer trabajo como desarrollador.

---

## 💡 ¿Qué hace este programa?

- Pide al usuario:
  - Su nombre
  - Edad
  - Si está inscrito en alguna materia
- Muestra un menú con opciones:
  - 📄 Ver perfil
  - 🧓 Editar edad
  - 🎓 Saber si es mayor de edad
  - 🚪 Salir del sistema

Todo el programa corre desde consola y mantiene el menú activo hasta que el usuario decide salir.

---

## 🧠 Tecnologías y conceptos usados

- Java básico
- Variables y tipos de datos
- Entrada de datos con `Scanner`
- Condicionales `if-else`
- Menú con `switch`
- Bucles con `do-while`
